# 2) **Backgrounds**

### **Landing Page Background**

* Class: `.landing-bg`

# 10) **Maps**

All maps are styled and initialized through `/js/map.js`.

### Tile source:

```js

```

### Marker style (dashboard):

```js
L.circleMarker([lat,lng], {
  radius: 7,
  fillColor: color,
  color: '#fff',
  weight: 1,
  fillOpacity: 0.9
})
```

### Heatmap:

Uses **leaflet.heat** plugin:

```js
L.heatLayer(points, { radius: 25, blur: 15 })
```
