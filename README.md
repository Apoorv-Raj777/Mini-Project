# Mini-Project
Sarthi: A Predictive Urban Safety Platform. Full-stack MERN/React Native &amp; Python app for crowdsourcing, analyzing, &amp; predicting public safety. Features predictive ML models (Scikit-learn) for unaudited areas, a "Safest Route" algorithm, &amp; an interactive React web dashboard with heatmaps. Integrates a Node.js API with a Python/Flask ML server.
